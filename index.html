Logging In

All users can immediately login, by clicking the Sign In button (top right) and entering their short username (eg bgray) and password

If you are logging in for the first time, this will create a user in the OpenProject system, and allow this user to be added to projects

Having a Play Around

There is both a Production and Test instance of this software available - if you want to try something out in a non-critical environment, do it in the Project-Test install which is updated nightly with data from Production

There are also some demo projects to get started with demo data we have prepared for you.

  • Demo project: to get an overview about classical project management.
  • Scrum project: to learn about Agile and Scrum project management.

Creating Projects

By default, you will not be able to create a Project, only participate in existing projects for which you have been added as a Member.

If you are interested in using the tool for Project Management, please contact IT (Bernard Gray) to organise training.

Upgrade to Enterprise edition

The OpenProject Enterprise edition builds on top of the Community edition. It includes Enterprise add-ons and professional support mainly aimed at organizations with more than 10 users that manage business critical projects with OpenProject.

Become a hero! Developers need to pay their bills, too. By upgrading to the Enterprise edition, you will be supporting this open source community effort and contributing to its development, maintenance and continuous improvement.

We deliver the confidence of a tested and supported enterprise-class project management software - with Open Source and an open mind.
